ALPHAFACTORFORGE 使用說明
這是什麼

抓取公開交易所行情（歷史 K 線 + 即時串流），讓你自定義買賣策略、做歷史回測，並用即時資料進行模擬交易（紙上交易，不涉及真實資金）。全部運算在你的瀏覽器本機完成。

① 兩種模式（右上切換）

回測：在歷史資料上測試策略，輸出淨報酬、勝率、最大回撤、夏普等績效與逐筆交易。改任何參數會自動重算（按鈕短暫變黃「回測中」）。

即時模擬：以即時行情運行虛擬帳戶（起始 $10,000）。可開自動交易讓策略依訊號進出場，或手動買賣。手續費 / 滑點 / 部位大小與回測同口徑。帳戶自動保存。

② 圖表操作（幣安級）

滾輪 = 以游標為錨縮放、按住拖曳 = 左右平移看更多歷史、雙擊 = 重置、滑鼠懸停 = 顯示該根 OHLC＋量＋漲跌（含完整年月日時間）。下方副圖為 RSI（含買 / 賣門檻虛線）。

③ 指標列（圖表下方）

每個指標是「開關 + 週期輸入框」：點 MA / EMA / BB / RSI / VOL 字樣切換顯示；旁邊的數字框可直接改週期（MA 有快 / 慢兩格、BB 有週期 / 倍數），改了圖表與回測同步更新。RSI 關閉時副圖會收起。

④ 範例策略（學習用）

策略編輯器頂部綠色「📚 範例策略」下拉，內建 6 組常見策略（均線交叉、RSI 反彈、MACD 動能、布林回歸、雙均線+RSI 濾網、突破+量能），選了即套用並回測，適合認識流程與三種編輯模式。

⑤ 策略編輯器 · 三種模式

參數：填週期並挑預設進出場訊號，最快速。

規則積木：組合多條件（全部成立才觸發），左選運算元、中選運算子、右填數字或另一運算元。

程式碼：寫 JavaScript 運算式，可用所有指標變數與前一根 prev 版本，打造獨創策略。

每個模式右上「? 本模式說明」有專屬教學；完整變數表與語法見專案內 STRATEGY_GUIDE.md。

⑥ 策略庫 + 參數掃描

策略庫：命名儲存目前策略、下拉切換、刪除（存於瀏覽器本機）。

參數掃描（最佳化）：選 1～2 個參數設範圍，跑遍所有組合（上限 256），用熱力圖（紅→黃→綠）顯示每格的最佳化指標，最佳格描黑框，可一鍵「套用最佳」。

⑦ 執行模型（讓回測貼近真實）

手續費 / 滑點：每邊計入；可點交易所建議值（幣安 / OKX…）一鍵帶入實際費率。部位大小：用權益的 % 下單。成交價：當根收盤 vs 下一根開盤。方向：做多 / 做空 / 多空反手。

停損 / 停利 %：對所有模式生效（0 = 關），以盤中高低判斷觸發。

Bar Magnifier 盤中精算：勾選後在每根 K 線底下抓更細子 K 線，精準判定停損 / 停利的先後順序（解決整根只有 OHLC、不知路徑的模糊）；狀態會回報「多少次與近似法不同」。區間過大會提示縮小日期。

⑧ 資料深度 + 回測區間

歷史深度：500 / 2000 / 5000 / 最大可得，多交易所分頁回拉（不再只有 500 根）。週期 1m～1d 七檔，要看盤中細節就調小週期。

資料來源：自動容錯或手動指定 Binance / OKX / Coinbase；綠點顯示目前實際用哪家。指定同一家或用「↓ JSON / ↑ 載入」凍結資料集，可得到完全可再現的回測。

回測區間：用起 / 訖日期選擇器或近 1 月 / 3 月 / 1 年 / 3 年快捷鈕，框定只測某段歷史（圖上區間外淡灰遮罩）。

樣本外驗證（70/30）：勾選後把區間切前 70% 樣本內 / 後 30% 樣本外分別回測對照，樣本外表現接近樣本內 = 較不過度擬合。

⑨ 專注數據模式 + 績效

圖表下方「▢ 專注數據」收起 K 線，放大顯示 23 項績效（淨報酬、夏普 / 索提諾 / 卡瑪、盈虧比、期望值、最大連勝連敗、平均 MFE/MAE…）、權益曲線疊買入持有、回撤 underwater 曲線，及逐筆 round-trip 明細（含 MFE / MAE）。

⑩ Bar Replay 逐根重播

圖表下方「▷ 逐根重播」開啟控制條，可播放 / 暫停 / 單步 / 拖曳，圖表只畫到當下那根，並即時顯示該根是進場 / 出場 / 無訊號——觀察策略訊號如何生成。

⑪ 匯出回測結果

在「專注數據」面板右上：↓ 績效報告 JSON（完整績效＋執行模型設定＋逐筆交易，可保存 / 再分析 / 分享）、↓ 交易明細 CSV（每筆 round-trip，可用 Excel 開）。圖表下方另有「↓ JSON / ↓ CSV」匯出原始 K 線資料集。

資料與免責

行情依序嘗試 Binance → OKX → Coinbase 三家公開 API，哪家連得上就用哪家。若都連不到，多半是網路 / 公司防火牆封鎖了交易所，換網路或手機熱點即可。本工具僅供研究與模擬，所有交易為虛擬，不構成投資建議。
策略編輯器 · 模式說明

最快上手。填入指標週期，再從下拉挑一組預設進場 / 出場訊號，即可回測經典技術策略。

可調參數

快線/慢線 MA、EMA、RSI 週期與超買/超賣門檻；「進階指標參數」另含 MACD(快/慢/訊號)、布林(週期/倍數)、ATR、隨機指標、量能 MA。

預設訊號

均線金叉/死叉、價格穿越 EMA、價格 vs 慢線、RSI 由超賣回升/超買回落、MACD 穿越訊號線、布林軌道觸碰、隨機指標回升/回落。

自由組合多條條件，全部成立 (AND) 才觸發。每條規則三欄：左運算元 · 運算子 · 右值

運算子

> 大於 · < 小於 · ≥ · ≤ · 上穿↑（由下突破）· 下穿↓（由上跌破）

右值可填數字（如 30）或另一個指標名稱（如 maSlow）。

範例
rsi < 30
maFast 上穿↑ maSlow
price > bbUpper

進場可疊多條（例：rsi < 30 且 macd 上穿↑ macdSignal），全部成立才進場。

最大彈性。進/出場各填一段 JavaScript 布林運算式（回傳 true 即觸發），系統以 return (你的運算式) 執行。

撰寫規則
  • 必須是單一運算式；不要寫 return、分號或多行。
  • 可用 > < >= <= === && || !、括號、算術、Math.*。
  • 每個變數是當根數值；加前綴 prev 代表前一根，用來偵測穿越。
  • 與 null 比較得 false（暖身不足），屬正常。語法錯會在編輯器下方紅字提示。
可用變數（當根，前綴 prev = 前一根）
price open high low volume
maFast maSlow ema rsi
macd macdSignal macdHist
bbUpper bbMid bbLower atr
stochK stochD volMa
慣用寫法
金叉：maFast > maSlow && prevMaFast <= prevMaSlow
RSI 回升：rsi > 30 && prevRsi <= 30
動能轉正：macdHist > 0 && prevMacdHist <= 0
突破+量能：price > bbUpper && volume > volMa * 1.5
複合：rsi < 40 && macd > macdSignal && price > ema

部位、手續費、滑點、停損停利、做多/做空請在「執行模型」設定，不要寫進運算式。完整版另見專案內 STRATEGY_GUIDE.md。
參數掃描 · 最佳化
在目前日期區間掃描參數組合，找出最佳值（上限 256 組）。掃描時忽略盤中精算以加速。
這是什麼 · 怎麼用

參數掃描幫你自動找出最佳參數。它把你選的參數依範圍逐一代入、各跑一次回測，再用熱力圖比較好壞，省去手動一個個試。

步驟
  1. 選 參數 X（要掃描的參數，如快線 MA），設起 / 迄 / 間距（例：5 到 20、間距 3 → 測 5,8,11,14,17,20）。
  2. 需要的話勾第二維 Y，同時掃兩個參數（產生二維熱力圖）。
  3. 選最佳化指標（要以什麼衡量「最佳」，如淨報酬、夏普、獲利因子）。
  4. 按執行掃描。每格 = 一組參數的結果，顏色越綠越佳、越紅越差，括號內為交易次數；最佳格描黑框。
  5. 按套用最佳把該組參數帶回策略。

注意：① 組合上限 256（超過請加大間距或縮範圍）。② 掃描忽略盤中精算以加速。③ 掃出的「最佳」務必再用樣本外驗證——在歷史上最好的參數，往往是過度擬合，未必對未來有效。
